🎙️ Live Speech Detection App
A real-time speech recognition web app that supports multiple languages, visualizes speech input, and features a stylish glassmorphic UI with a drifting ocean background.


🎬 Demo

🚀 Features
🎤 Real-time speech recognition

🌐 Language selector – supports English, Spanish, Hindi, Telugu, and French

📊 Live audio visualizer

🌊 Dynamic ocean background

🧊 Glassmorphic UI with 3D hover effects

✨ Bonus Technologies
CSS Keyframe Animations – for the ocean drift and live bar visualizer.

Glassmorphism – using backdrop-filter: blur(8px) and transparency effects to create a "frosted glass" card look.

3D Transformations – using CSS transform-style: preserve-3d to make the card tilt on hover.

Quick Visual Summary:
Frontend: HTML5 + CSS3 + JavaScript (no frameworks like React/Vue)

Styling & UI: Bootstrap 5 + custom CSS (Glassmorphism & Animations)

APIs: Native Web Speech API (no external speech services needed)

Development Tools: BrowserSync for hot-reloading
